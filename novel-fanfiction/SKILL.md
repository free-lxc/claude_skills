# Novel Fanfiction - 小说二创工具

## 概述

Novel Fanfiction 是一个强大的小说二次创作辅助工具，能够基于提供的小说内容进行多种形式的创意改编和续写。无论是同人创作、风格练习，还是探索新的叙事可能性，这个工具都能提供专业的创作支持。

## 核心功能

### 1. 剧情续写 (Continue Mode)

基于原作设定和情节走向，续写新的章节内容。

**使用方法：**
```
使用 novel-fanfiction skill，模式：continue
原作内容：[粘贴小说片段或简介]
续写要求：[描述希望续写的方向或具体要求]
```

**特点：**
- 保持原作的人物性格和说话方式
- 延续原作的世界观设定
- 自然衔接原作情节
- 可指定续写篇幅（章节数/字数）

---

### 2. 情节改写 (Rewrite Mode)

对指定章节或情节进行重新创作，改变叙事结构或呈现方式。

**使用方法：**
```
使用 novel-fanfiction skill，模式：rewrite
原作内容：[粘贴需要改写的片段]
改写要求：[说明改写的目标，如更紧凑、更幽默等]
```

**改写维度：**
- 叙事节奏调整（加快/放慢）
- 悬念设置增强
- 对话优化
- 描写密度调整
- 结构重组（倒叙、插叙等）

---

### 3. 风格转换 (Style Mode)

将原作内容转换为不同的文学风格。

**使用方法：**
```
使用 novel-fanfiction skill，模式：style
原作内容：[粘贴小说片段]
目标风格：[指定风格，如古龙风、翻译腔、网络文学风等]
```

**支持的风格：**
- 古龙风（短句、留白、意境）
- 金庸风（历史厚重、武侠底蕴）
- 翻译腔（欧化句式、倒装结构）
- 网络文学风（轻快、吐槽、梗多）
- 意识流（内心独白、联想跳跃）
- 纪实风格（冷峻客观、细节详实）
- 童话风格（纯净、寓言式）
- 黑色幽默（荒诞、讽刺）

---

### 4. 视角转换 (POV Mode)

从不同角色的视角重述相同的故事。

**使用方法：**
```
使用 novel-fanfiction skill，模式：pov
原作内容：[粘贴原片段]
目标视角：[指定角色或叙事视角]
```

**视角选项：**
- 第一人称（我）
- 第三人称限制视角（聚焦某角色）
- 第三人称全知视角
- 第二人称（你）
- 特定角色视角
- 反派/配角视角
- 群像视角

---

### 5. 情节拓展 (Expand Mode)

在原有情节基础上增加细节、心理描写和场景刻画。

**使用方法：**
```
使用 novel-fanfiction skill，模式：expand
原作内容：[粘贴需要拓展的片段]
拓展方向：[说明需要加强的方面，如心理描写、环境细节等]
```

**拓展维度：**
- 心理活动深化
- 环境氛围营造
- 动作细节分解
- 对话层次丰富
- 背景故事补充
- 感官体验增强（视觉、听觉、触觉等）

---

### 6. 平行宇宙 (AU Mode)

保留核心角色设定，改变世界观背景进行创作。

**使用方法：**
```
使用 novel-fanfiction skill，模式：au
原作角色：[描述核心角色特征]
新背景设定：[描述新的世界观或背景]
创作要求：[说明希望创作的情节]
```

**AU类型：**
- 现代AU（古代角色到现代）
- 校园AU
- 职场AU
- 末世AU
- 科幻AU
- 奇幻AU
- 互换身份AU

---

### 7. 假设性场景 (What If Mode)

探索关键情节的不同可能性分支。

**使用方法：**
```
使用 novel-fanfiction skill，模式：whatif
原作转折点：[描述原作中的关键选择或事件]
假设条件：[说明"如果...会怎样"的假设]
```

**示例假设：**
- "如果主角在那个关键时刻选择了另一条路"
- "如果反派没有被击败"
- "如果两人提前相遇"
- "如果某个重要角色没有死亡"

---

### 8. 跨作品联动 (Crossover Mode)

将不同作品的世界观和角色进行融合创作。

**使用方法：**
```
使用 novel-fanfiction skill，模式：crossover
作品A：[描述第一个作品的世界观和角色]
作品B：[描述第二个作品的世界观和角色]
联动设定：[说明两个世界如何连接]
```

---

## 高级功能

### 个性化设定

可以在输入中指定以下参数来个性化创作结果：

- **语言风格**：正式/口语/古典/现代
- **情感基调**：轻松/沉重/温馨/紧张
- **叙事节奏**：紧凑/舒缓
- **篇幅要求**：字数/段落数
- **内容偏好**：更注重对话/心理/动作/描写

### 多段处理

对于长篇小说，可以分批处理：

```
使用 novel-fanfiction skill
模式：continue
原作内容：[第1-3章内容]
续写要求：续写第4章，保持悬疑感，2000字左右

[待处理完成后再继续]
使用 novel-fanfiction skill
模式：continue
原作内容：[第1-4章内容]
续写要求：续写第5章，揭示部分线索，2000字左右
```

## 使用示例

### 示例1：续写悬疑小说

```
使用 novel-fanfiction skill，模式：continue

原作内容：
侦探推开了那扇吱呀作响的木门，房间里空无一人，只有一盏台灯在角落里忽明忽暗。桌上一杯咖啡还冒着热气，显然主人刚刚离开。他在地板上发现了一张照片，照片背面写着一行字："真相在地下室。"

续写要求：
1. 续写侦探前往地下室的情节
2. 保持悬疑紧张的氛围
3. 字数约1500字
4. 在章节末尾留下新的悬念
```

### 示例2：风格转换

```
使用 novel-fanfiction skill，模式：style

原作内容（现代言情风）：
他看着她，眼神温柔地说："你今天真漂亮。"她听了脸一下子红了起来，小声说："你净会哄人开心。"

目标风格：古龙风

转换要求：
保持原意，但使用古龙的短句风格和留白技巧
```

### 示例3：视角转换

```
使用 novel-fanfiction skill，模式：pov

原作内容（主角视角）：
我看着他对我的挑衅，心中的怒火在燃烧。这个人在众人面前羞辱我，我绝不会就此罢休。我握紧了拳头，准备给他一个教训。

目标视角：旁观者视角

转换要求：
从现场旁观者的角度描述这场冲突，表现出对局势的紧张和担忧
```

### 示例4：情节拓展

```
使用 novel-fanfiction skill，模式：expand

原作内容：
他们在雨夜分别，彼此都知道这次可能再也见不到了。她转身离开，没有回头。

拓展要求：
1. 深化两人的内心活动
2. 增加环境描写（雨、街景、氛围）
3. 细化离别的动作和神态
4. 拓展至500字左右
```

### 示例5：平行宇宙AU

```
使用 novel-fanfiction skill，模式：au

原作角色：
诸葛亮：三国时期蜀汉丞相，智慧超群，运筹帷幄
刘备：蜀汉开国皇帝，仁德爱民，善于用人

新背景设定：
现代职场AU
- 诸葛亮：某知名科技公司CTO，技术天才
- 刘备：公司创始人兼CEO，有远见但技术一般

创作要求：
写一场公司面临收购危机的会议场景，展现两人如何配合应对
```

## 注意事项

1. **内容完整性**：提供足够的原作信息，包括人物关系、世界观设定、相关情节
2. **明确需求**：清晰说明想要达到的效果和具体要求
3. **尊重原作**：除非明确要求改写，否则续写应尽量保持原作风格和设定
4. **分段处理**：长篇小说建议分章节逐步处理
5. **反馈调整**：可以基于第一次输出提出修改意见，进行迭代优化

---

## 适用场景

- **同人创作**：基于喜欢的作品进行二次创作
- **写作练习**：通过改写和视角转换提升写作技巧
- **创意拓展**：为卡住的剧情寻找新的可能性
- **风格实验**：尝试不同的叙事风格和语言方式
- **情节优化**：对初稿进行润色和改进
- **教学演示**：展示同一情节的不同处理方式
